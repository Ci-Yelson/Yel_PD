# Yel_PD


基于有限元法实现10万+顶点规模的复杂拓扑结构模型的实时交互仿真

仿真算法参考：["Hyper-Reduced Projective Dynamics"](https://github.com/RotateMe/HyperReducedProjectiveDynamics) [2018; Christopher Brandt, Elmar Eisemann, Klaus Hildebrandt]

---

- Bear: 117,908 vertices; 238,062 faces; 461,070 tetrahedrons.

![](./imgs/Bear.gif)

- Shoe: 177,638 vertices; 363,238 faces; 627,134 tetrahedrons.
